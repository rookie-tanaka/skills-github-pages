MOBO ExtraLight (font-weight: 200)

軽やかで繊細な印象のExtraLight
本文や長文に適した軽い印象のウェイトです。読みやすさを重視する場面で活躍します。
.mobo-extralight { font-family: 'MOBO', sans-serif; font-weight: 200; }

MOBO Regular (font-weight: 400)

標準的で使いやすいRegular
最も汎用性が高く、どんな場面でも使いやすい標準的なウェイトです。
.mobo-regular { font-family: 'MOBO', sans-serif; font-weight: 400; }

MOBO SemiBold (font-weight: 600)

しっかりとした存在感のSemiBold
見出しや強調したい部分に最適。適度な太さで視認性も抜群です。
.mobo-semibold { font-family: 'MOBO', sans-serif; font-weight: 600; }

MOBO Bold (font-weight: 700)

インパクト抜群のBold
タイトルやロゴに最適。角丸処理が最も効果的に表現されるウェイトです。
.mobo-bold { font-family: 'MOBO', sans-serif; font-weight: 700; }

MOBOフォントのテスト

あなたのサイトでの使用方法

1. フォントファミリーの指定

/* 基本の使用方法 */ body { font-family: 'MOBO', 'Hiragino Sans', 'Yu Gothic Medium', sans-serif; } /* 各ウェイトの使用 */ .title { font-family: 'MOBO'; font-weight: 700; } .heading { font-family: 'MOBO'; font-weight: 600; } .body-text { font-family: 'MOBO'; font-weight: 400; } .caption { font-family: 'MOBO'; font-weight: 200; }

2. 既存のクラスを更新

/* h2タイトルにMOBOフォントを適用 */ h2 { font-family: 'MOBO', sans-serif; font-weight: 700; /* Bold */ } /* フッターのテキストにMOBOフォントを適用 */ .wave-text { font-family: 'MOBO', sans-serif; font-weight: 600; /* SemiBold */ }

3. 新しいユーティリティクラス

/* 便利なクラスを追加 */ .font-mobo { font-family: 'MOBO', sans-serif; } .font-mobo-light { font-family: 'MOBO', sans-serif; font-weight: 200; } .font-mobo-regular { font-family: 'MOBO', sans-serif; font-weight: 400; } .font-mobo-semibold { font-family: 'MOBO', sans-serif; font-weight: 600; } .font-mobo-bold { font-family: 'MOBO', sans-serif; font-weight: 700; }
